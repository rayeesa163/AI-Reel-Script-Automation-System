# AI Reel Script Automation System

## Overview

This project is an AI-powered short-form content automation workflow built using:

* n8n
* Google Gemini AI

The workflow automates one of the biggest bottlenecks in content operations:

* scripting
* ideation
* hook generation
* CTA generation

## Workflow

User Input
→ AI generates viral hook
→ AI generates reel script
→ AI generates CTA

## Features

* AI-powered reel generation
* Viral hook generation
* Reusable workflow templates
* Multi-niche scalability
* Reduced manual scripting effort

## Tech Stack

* n8n
* Google Gemini
* AI Agent workflows

## Scaling Logic

The workflow is reusable across multiple niches by changing:

* topic
* audience
* content style

This allows one operator to manage multiple content channels efficiently.
